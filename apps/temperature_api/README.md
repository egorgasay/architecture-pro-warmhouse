# Temperature API

Flask приложение с чистой архитектурой для работы с температурными данными.

## Архитектура

- **Controllers** - обработка HTTP запросов
- **Services** - бизнес-логика
- **Repositories** - слой доступа к данным

## Установка и запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
python app.py
```

## API Endpoints

### GET /temperature
Возвращает температуру для заданной локации.

**Параметры:**
- `location` (строка): Локация
